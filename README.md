
# University Library System

A web-based University Library Management System developed as a university project using React, TypeScript, and Vite.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Acknowledgements

This project utilizes several open-source libraries, including **shadcn/ui** (MIT License), to build a modern and responsive user interface.
